Loops

A loop is a block of code that repeats itself until a given condition is fulfilled. There are a few types of loops:

  • For Loops
  • While Loops
  • Do While Loops

Functions

A function is a block of code that can be used by reference. The two examples above were called using functions.

Variables

Variables were also used in the previous examples. A variable is a stored value, whether that's a string of characters or a number.

Parameters

A parameter is a variable that's used in a function to determine which value to use.

Arrays

Arrays are containers that hold multiple variables. Each variable is numbered and can be accessed according to that number.

Associative Arrays

Associative arrays are like normal arrays but instead of being numbered they have keys to each variable.
